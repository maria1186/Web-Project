## Weather app with Vanilla JavaScript

Version 1: September 2022 year

### Version 1:

Two API requests are made in the application: for the 5-day weather forecast. 
The icons provided in the API are used, as well as four of my choice displayed: 
Humidity, Wind Speed, Clouds and Sunset.
The [Openweathermap](https://openweathermap.org/api) API is used. 


Version 2: November 2025 year

### Version 2:

Open-Meteo partners with national weather services to bring you open data with high resolution,
ranging from 1 to 11 kilometres. Our powerful APIs intelligently select the most suitable
weather models for your specific location, ensuring accurate and reliable forecasts.
I changed the weather service so I don't need to register.
The [Open-Meteo](https://open-meteo.com/en/docs) API is used.
